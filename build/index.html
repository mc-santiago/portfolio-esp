Echale un vistazo a algunos de mis proyectos de Github.

No solo incluyo la descripción y detallo las tecnologías utilizadas, sino que puedes hacer click en el título de cada proyecto para ver el resultado final.

JavaScript Portfolio

Creé este portfolio como una manera de poner en práctica los ejercicios que tuve que completar para la obtención del certificado de JavaScript Algorithms and Data Structures. Incluye una sección, paso a paso, que explica el código a nivel #CodeNewbie.

Tecnologías utilizadas: HTML5UP(modificado) y Vanilla JavaScript.

EarnEat Dashboard

Dashboard totalmente responsivo, creado para el uso de los empleados de ApetEat. Trabajé con un equipo de tres en donde todas compartiamos tareas tanto de front-end como de back-end, pero yo era la encargada de que todo saliera bien en el front-end.

Tecnologías utilizadas: MySQL, NodeJS, ReactJS, SemanticUI, ExpressJS, PassportJS, entre otros.

Battle Spaceships

En este proyecto trabajé en equipo con una integrante más. Este juego tipo space invaders nos hizo ganar un Hackathon Game Challenge.

Tecnologías utilizadas: Phaser3 y CSS.

Random Quote Generator

Esta página selecciona citas de Los Simpsons (con su propia imagen) de forma aleatoria. La información es obtenida de una API al hacer click en el botón.

Tecnologías utilizadas: Axios, ReactJS, CSS y el API thesimpsonsquoteapi.glitch.me/.
